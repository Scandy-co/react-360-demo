# react 360 demo

A demo showing a 3D model in space using React 360.

## Setup

```
yarn
```

or

```
npm install
```

## Running

```
yarn start
```

or

```
npm start
```

Then nativgate to you `http://$IP_ADDRESS:8081/index.html` from your VR headset or `http://localhost:8081/index.html` from you desktop
